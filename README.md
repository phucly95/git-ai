# Git AI

Angular 20 application with AI-powered Git assistant features.

## Description

Git AI is a modern web application built with Angular 20 that provides intelligent Git workflow assistance. The application leverages artificial intelligence to help developers manage their Git repositories more efficiently, offering smart suggestions, automated workflows, and enhanced version control capabilities.

## Getting Started

### Prerequisites
- Node.js (18+)
- npm (8+)
- Angular CLI (20+)
- Git.

### Installation
```bash
npm install
```

### Development
```bash
npm start
```

### Build
```bash
npm run build
```

### Testing
```bash
npm run testing
```
