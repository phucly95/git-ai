# Git AI

Angular 20 application with AI-powered Git assistant features.

## Getting Started

### Prerequisites
- Node.js (18+)
- npm (8+)
- Angular CLI (20+)
- Git.

### Installation
```bash
npm install
```

### Development
```bash
npm start
```

### Build
```bash
npm run build
```

### Testing
```bash
npm run testing
```
