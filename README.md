# Math Utilities

Simple Python module for basic math operations.

## Functions

- `add_two_numbers(a, b)` - Add two numbers
- `multiply_two_numbers(a, b)` - Multiply two numbers  
- `divide_two_numbers(a, b)` - Divide two numbers
- `subtract_two_numbers(a, b)` - Subtract two numbers

## Usage

```python
from sum import add_two_numbers, multiply_two_numbers

# Addition
result = add_two_numbers(5, 3)  # Returns 8

# Multiplication  
product = multiply_two_numbers(4, 2)  # Returns 8
```
