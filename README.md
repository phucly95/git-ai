# Git AI

Angular 20 application with AI-powered Git assistant features.

## Description

Git AI is an Angular 20 application that provides AI-powered Git workflow assistance for developers.
This application streamlines Git operations through intelligent automation and smart suggestions.
It helps developers manage repositories more efficiently with AI-driven insights and recommendations.

Key features include automated commit message generation, intelligent branch management, and smart merge conflict resolution.
The application leverages machine learning algorithms to analyze code patterns and suggest optimal Git workflows.
Built with modern Angular architecture, it offers a responsive user interface and seamless integration with existing development tools.
Developers can benefit from predictive analytics that help identify potential issues before they occur.
The system also provides comprehensive reporting and analytics to track repository health and team productivity.

## Getting Started.

### Prerequisites
- Node.js (18+)
- npm (8+)
- Angular CLI (20+)
- Git.

### Installation
```bash
npm install
```

### Development
```bash
npm start
```

### Build
```bash
npm run build
```

### Testing
```bash
npm run testing
```
