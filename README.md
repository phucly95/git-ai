# Git AI

Angular 20 application with AI-powered Git assistant features.

## Getting Started

### Prerequisites
- Node.js (version 18 or higher)
- npm (version 8 or higher)
- Angular CLI (version 20 or higher)
- Git

### Installation
```bash
npm install
```

### Development
```bash
npm start
```

### Build
```bash
npm run build
```

### Testing
```bash
npm run testing
```
