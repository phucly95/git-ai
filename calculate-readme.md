# Calculate.js

Simple JavaScript utility for mathematical operations.

## Description

This module provides basic arithmetic functions for JavaScript applications.
It offers a simple and efficient way to perform mathematical calculations in your code.
The functions are designed to handle both integer and floating-point numbers safely.
Each function includes comprehensive documentation and examples for easy implementation.
Perfect for developers who need reliable mathematical operations in their projects.

## Usage

```javascript
const result = sum(5, 3);
console.log(result);
```

## Functions

- `sum(a, b)` - Adds two numbers together
